# LaborVsAiCompute
Datascience Dashboard on "Autonomous Agentic Workflows" vs "Human Labor"

## The CSV File
The CSV file has the following data: 
### Rows
1. 1201 rows
### Columns
1. ID 
2. Industry 
    (80 in Finance, Legal, Healthcare, MLOps, Creative, Education, Manufacturing, Retail, Logistics, Energy, Real Estate, Insurance, Consulting, Government, and Telecom)
3. Human_Labor_Cost_hr
4. Tokens_per_Human_Hour
5. Inference_Cost_2026
6. Agent_Labor_Equivalent_Cost
7. Substitution_Elasticity
8. AI_Augmentation_Factor
9. Automation_Risk_Index
10. Hardware_CapEx_Sensitivity
11. Regulatory_Moat
12. Substitution_Year_Est
